# Drone Navigation Assignment

## Overview

This project is a Unity-based drone simulation developed as part of the selection task. It features a 3D open-world environment where the user can control a drone using realistic flight controls. The project includes multiple camera views, UI controls, and a real-time flight status display.

---

## Features

- 3D Open World Environment
- Drone Flight Controller
- Stable Hovering
- First-Person (FPV) Camera
- Third-Person Camera
- Arm / Disarm Drone
- Real-Time Flight HUD
  - Speed
  - Altitude
  - Arm Status
  - Camera Mode
  - Compass

---

## Controls

| Key | Action |
|------|--------|
| W | Pitch Forward |
| S | Pitch Backward |
| A | Roll Left |
| D | Roll Right |
| Q | Yaw Left |
| E | Yaw Right |
| Space | Throttle Up |
| Left Ctrl | Throttle Down |

---

## UI Buttons

- Arm Drone
- Disarm Drone
- Switch to First-Person View
- Switch to Third-Person View

## How to Run

1. Clone the repository.

```
git clone https://github.com/chandana200420/Drone-Assignment.git
```

2. Open the project in Unity.
3. Open the main scene.
4. Press **Play**.
5. Click **Arm** to activate the drone.
6. Use the keyboard controls to fly.
